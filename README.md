## OSINT-SAN PRO

#### Framework направленный на получение данных и деанонимизацию пользователей. В инструменте используется большое количество API.

Для получение PRO версии напишите в лс в телеграм: https://t.me/satana666mx. Или в чат паблика в telegram: https://t.me/osint_san_framework

Напомню что инструмент еженедельно обновляется.
